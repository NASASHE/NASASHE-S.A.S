Domestika Downloader

Guía rápida para la herramienta de descarga y despliegue en GitHub Pages.

Descarga local

  1. Edita index.js y completa course_url, subtitle_lang, las cookies y las credenciales.
  2. Asegúrate de tener N_m3u8DL-RE.exe y ffmpeg instalados en la misma carpeta.
  3. Instala dependencias con npm install y ejecuta npm start para iniciar la descarga.

Despliegue en GitHub Pages

  1. Habilita GitHub Pages en la configuración del repositorio apuntando al flujo de trabajo de GitHub Actions.
  2. El contenido de la carpeta public (este archivo) se publicará automáticamente en la rama de Pages.
  3. Personaliza este HTML con la información o interfaz que necesites mostrar.

El flujo de trabajo se ejecuta en cada push a main y también puede lanzarse manualmente.
